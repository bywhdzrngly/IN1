

| 技术类型          | 特点                                  | 适合你们的场景？|
|-------------------|---------------------------------------|-------------------------|
| 原生 HTML+CSS+JS  | 基础、灵活，但代码量大、易混乱        | 极简版聊天App（够用）|
| React 前端        | 组件化、数据驱动，代码规整，适合复杂界面 | 想做功能多的聊天App（比如加表情包、文件传输） |
| Vue 前端          | 上手比 React 简单，语法更贴近原生 JS  | 新手优先（和 React 功能类似，学习成本更低） |
